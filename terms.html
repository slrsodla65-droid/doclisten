1. 서비스 내용

DocListen는 사용자가 업로드한 PDF를 화면에 표시하고, 문단 단위로 한국어 음성 재생을 제공하는 웹 서비스입니다.

2. 베타 서비스 고지

현재 서비스는 베타 단계입니다. 기능, 음성 품질, 처리 속도, 파일 보관 정책은 정식 출시 전 변경될 수 있습니다.

3. 사용자 책임

  • 본인이 이용 권한을 가진 PDF만 업로드해야 합니다.
  • 저작권, 개인정보, 영업비밀이 포함된 문서 업로드 시 사용자가 책임을 집니다.
  • 불법 콘텐츠, 악성 파일, 타인의 권리를 침해하는 문서는 업로드할 수 없습니다.

4. 서비스 제한

시스템 부하, 악용 방지, 법적 문제, 운영상 필요가 있을 경우 일부 기능 또는 사용량이 제한될 수 있습니다.

5. 유료 베타 이용기간 및 환불

  • Beta Pro는 카카오톡 수동 신청과 입금 확인 후 1회용 베타 코드로 활성화됩니다.
  • 유료 베타 이용기간은 결제 확인일로부터 30일입니다.
  • 베타 코드를 아직 사용하지 않은 경우 결제 확인 후 7일 이내 환불 요청이 가능합니다.
  • 코드 사용 후에는 서비스 장애로 정상 이용이 불가능한 경우를 제외하고 환불이 제한될 수 있습니다.
  • 환불 요청은 Google 로그인 이메일과 입금자명을 포함해 카카오톡 오픈채팅으로 접수합니다.

6. 면책

PDF 텍스트 추출과 음성 합성은 문서 구조에 따라 오류가 발생할 수 있습니다. 중요한 계약서, 법률, 의료, 투자 판단에는 원문 확인이 필요합니다.
