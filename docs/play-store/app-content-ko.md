# Play Console 앱 콘텐츠 답변 초안

## 개인정보처리방침
- URL: https://doclisten.app/privacy.html

## 앱 액세스
- 제한된 기능 또는 로그인 요구: 없음
- 설명: Android 앱 모드에서는 로그인 없이 PDF 업로드와 듣기 기능을 사용할 수 있습니다.

## 광고
- 광고 포함 여부: 예
- 설명: 운영비 충당을 위해 Google 광고가 표시될 수 있으며, 개인정보처리방침에 광고 쿠키/Google 광고 정책 내용을 고지했습니다.

## 타겟층 및 콘텐츠
- 권장 대상: 일반 사용자, 학생, 직장인
- 민감 콘텐츠: 없음
- 어린이 대상 앱: 아니요
- 13세 미만을 주요 대상으로 하지 않음

## 콘텐츠 등급 설문 예상
- 폭력성: 없음
- 선정성: 없음
- 혐오/차별: 없음
- 약물/도박: 없음
- 사용자 생성 콘텐츠: 없음. 사용자가 본인 PDF를 선택해 듣는 기능만 제공
- 온라인 상호작용: 제한적. PDF 텍스트 처리와 음성 생성을 위한 서버 통신

## 데이터 보안 핵심 문구
- 사용자가 선택한 PDF 텍스트는 문단 표시와 음성 생성을 위해 처리됩니다.
- 파일/텍스트를 광고주에게 판매하지 않습니다.
- 전송은 HTTPS로 보호됩니다.
- 삭제 문의는 https://doclisten.app/contact.html 에서 접수합니다.
