# Play Console 데이터 보안 답변 초안

> 실제 Play Console 화면 문항은 Google이 수시로 바꾸므로, 아래 내용을 기준으로 선택지를 맞춰 입력합니다.

## 데이터 수집/처리 개요
DocListen Android 앱은 사용자가 선택한 PDF를 화면에 렌더링하고 문단을 음성으로 듣기 위해 PDF 텍스트를 처리합니다. Android 앱 모드에서는 로그인 없이 기본 PDF 듣기 기능을 사용할 수 있습니다.

## 데이터 암호화
- 전송 중 암호화: 예. HTTPS(`https://doclisten.app`) 사용.

## 데이터 삭제 요청
- 예. 개인정보처리방침/문의 페이지를 통해 삭제 요청 가능.
- URL: https://doclisten.app/contact.html

## 수집 가능 데이터 유형

### 파일 및 문서
- PDF 파일명
- PDF에서 추출된 텍스트
- 목적: 앱 기능 제공, PDF 문단 표시, 음성 생성
- 필수 여부: PDF 듣기 기능 사용 시 필요
- 공유 여부: 서비스 제공을 위한 서버/인프라 처리 외 판매하지 않음

### 앱 활동
- PDF 업로드, 듣기 시도, 문단 사용 흐름 같은 기본 이벤트
- 목적: 기능 제공, 오류 개선, 서비스 안정성

### 앱 정보 및 성능
- 오류 로그, 기본 접속 로그
- 목적: 앱 안정성, 장애 대응

### 기기 또는 기타 ID / 광고 관련 데이터
- Google 광고 또는 웹뷰/분석성 서비스가 제한적으로 사용할 수 있음
- 목적: 광고 표시, 광고 성과 측정, 부정 사용 방지
- Play Console의 “광고 포함” 항목은 보수적으로 “예” 선택 권장

## 로그인/계정
- Android 앱 기본 기능은 로그인 없이 사용 가능.
- 웹 서비스에는 Google 로그인 기반 사용량 관리 기능이 존재할 수 있으나, Android 앱 모드에서는 기본 PDF 듣기 흐름에서 필수 아님.

## 민감정보
- 건강, 금융, 위치, 연락처, 사진/동영상 라이브러리 전체 접근은 앱 기능상 요구하지 않음.
- 사용자가 직접 선택한 PDF만 파일 선택기를 통해 처리.

## 권장 Play Console 선택 요약
- 앱에서 데이터를 수집하나요?: 예
- 데이터가 암호화되어 전송되나요?: 예
- 사용자가 데이터 삭제를 요청할 수 있나요?: 예
- 앱에 광고가 포함되어 있나요?: 예
- 앱 접근 권한/로그인 필요?: 아니요. 기본 기능은 로그인 없이 사용 가능
