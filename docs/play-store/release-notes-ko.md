# Play Store 릴리스 노트

## 1.0.1 (versionCode 2)

비공개 테스트 전 사용자 흐름을 개선한 안정화 릴리스입니다.

- 일시정지 상태에서 `이어듣기`와 `재개` 모두 같은 위치에서 계속 재생
- 웹에서 로그인 전 PDF 업로드를 막고 먼저 로그인하도록 안내
- Android 앱 모드의 불필요한 Google 로그인 영역과 안내 문구 제거
- PDF 추출 문장의 문장부호·영문 약어·일부 띄어쓰기 보정
- 서버 음성에도 정리된 낭독 문장 적용
- 모바일 하단 재생 패널 크기 축소 및 버튼 배열 개선
- 고해상도 Android 화면에서 PDF 글자를 선명하게 렌더링하고 페이지 전체 폭 맞춤

## 1.0.0 (versionCode 1)

첫 Android 테스트 릴리스입니다.

- PDF 업로드 및 페이지 렌더링 지원
- 문단 하이라이트와 문단 단위 듣기 지원
- 듣기, 일시정지, 이전/다음 문단 이동 지원
- 음성 속도 조절 지원
- Android 앱 모드에서 로그인 없이 PDF 듣기 가능
- 외부 결제/유료 CTA는 앱 모드에서 숨김 처리
- 개인정보처리방침 및 문의 URL 제공

## 내부 테스트 확인
- Android 15 Google APIs 에뮬레이터에서 앱 실행 확인
- 테스트 PDF 업로드 확인
- 문단 하이라이트 확인
- 오디오 포커스/AAudio 재생 로그 확인
- `npm run test` 24/24 통과
- `./gradlew :app:bundleRelease` 성공
