## 짧은 설명
PDF를 업로드하고 문단별로 보면서 바로 들을 수 있는 PDF 듣기 앱입니다.

## 상세 설명
DocListen은 긴 PDF 문서를 눈으로만 읽기 어려운 사용자를 위한 PDF 듣기 앱입니다. 강의자료, 리포트, 논문 초안, 업무 문서처럼 텍스트가 많은 PDF를 업로드하면 문서 화면을 보면서 현재 문단을 하이라이트하고, 선택한 위치부터 음성으로 들을 수 있습니다.

주요 기능:
- PDF 업로드 및 페이지 보기
- 문단 단위 텍스트 추출과 현재 문단 하이라이트
- 듣기, 일시정지, 이전/다음 문단 이동
- 음성 속도 조절
- 모바일 화면에 맞춘 PDF 읽기/듣기 컨트롤
- 개인정보처리방침과 문의 페이지 제공

추천 대상:
- 출퇴근길이나 이동 중에 PDF를 듣고 싶은 사용자
- 긴 강의자료와 리포트를 반복 확인해야 하는 학생·연구자
- 업무 문서를 빠르게 훑고 싶은 직장인
- PDF 문서를 접근성 관점에서 듣고 싶은 사용자

주의사항:
텍스트 선택이 가능한 PDF에 가장 적합합니다. 스캔 이미지 PDF, 복잡한 표 중심 문서, 이미지로만 구성된 문서는 텍스트 추출과 음성 품질이 떨어질 수 있습니다.
