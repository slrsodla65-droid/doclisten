# DocListen 모바일 앱 테스트 플랜

## 기본 실행
1. 앱 실행
2. 홈 화면이 표시되는지 확인
3. 공개 가이드/개인정보/문의 페이지 접근 확인

## PDF 듣기
1. PDF 업로드 버튼 선택
2. 텍스트 PDF 선택
3. PDF 페이지가 렌더링되는지 확인
4. 듣기 버튼 실행
5. 문단 하이라이트와 음성 재생 확인
6. 일시정지, 이전/다음 문단, 속도 조절 확인

## 계정/사용량
1. Google 로그인 시작
2. 로그인 완료 후 사용량 표시 확인
3. 무료 체험 한도 안내 확인
4. 로그아웃 확인

## 네트워크/오류
1. 네트워크 불안정 상태에서 안내 메시지 확인
2. 스캔 PDF 업로드 시 한계 안내 확인
3. 서버 TTS 실패 시 브라우저 음성 fallback 확인

## 심사 제출 전 필수
- 실제 iPhone에서 PDF 선택/재생 확인
- iPad 레이아웃 확인
- 앱 종료 후 재실행 확인
- 개인정보처리방침/문의 링크 정상 확인
