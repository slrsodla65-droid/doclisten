# DocListen App Store 심사 노트 초안

## 앱 개요
DocListen은 사용자가 직접 선택한 PDF 문서를 업로드하고, 문단 단위로 화면을 보면서 한국어 음성으로 들을 수 있게 돕는 생산성/접근성 앱입니다. 긴 문서, 강의자료, 리포트, 논문 초안을 눈으로만 읽기 어려운 사용자를 위해 문단 하이라이트, 이전/다음 문단 이동, 속도 조절, 이어듣기 흐름을 제공합니다.

## 핵심 기능
- PDF 파일 선택 및 문서 화면 표시
- PDF 텍스트 문단 추출 및 문단별 터치 선택
- 한국어 자연 낭독 TTS 재생
- 재생/일시정지/이전 문단/다음 문단
- 읽는 문단 하이라이트
- 무료 체험 사용량 안내 및 계정 기반 사용량 저장
- 파일 처리 정책, 개인정보처리방침, 문의 경로 제공

## 심사 계정/접근
- 앱은 기본 기능 확인을 위해 공개 사이트와 서버 API가 필요합니다.
- Google 로그인 없이도 UI와 공개 가이드는 확인할 수 있습니다.
- 계정 기반 무료 체험/사용량 확인이 필요한 경우 심사용 Google 계정 또는 테스트 계정을 별도로 제공해야 합니다.

## 서버/API
- 운영 URL: https://doclisten.app/
- 앱은 HTTPS 서버와 통신합니다.
- PDF 파일은 사용자가 직접 선택하며, 정책 페이지에서 파일 처리 방식을 안내합니다.

## 결제 관련
현재 iOS/Android 앱 셸에서는 외부 결제/카카오톡 유료 전환 CTA를 숨기고 무료 PDF 듣기 유틸리티로 동작하도록 처리했습니다. 향후 앱 내 유료 기능을 제공하려면 Apple In-App Purchase(IAP) 적용 후 다시 제출해야 합니다.

## 심사 리스크 및 대응
Apple App Review Guideline 4.2는 단순 웹사이트 재포장 앱을 제한합니다. DocListen은 PDF 업로드, 문단 추출, 음성 재생, 문단 하이라이트, 이어듣기라는 앱형 유틸리티를 핵심 기능으로 설명해야 합니다. 제출 전 네이티브 파일 열기/공유 수신, 온보딩, 앱 아이콘/스크린샷을 보강하면 승인 가능성이 높아집니다.
