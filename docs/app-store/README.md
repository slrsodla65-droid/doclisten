# DocListen Mobile Release Kit

이 폴더는 DocListen을 iOS App Store / Android 배포용 앱으로 준비하기 위한 자료입니다.

- `listing-ko.md`: 앱스토어 등록 문구 초안
- `review-notes-ko.md`: App Review 제출 노트 초안
- `release-checklist-ko.md`: 출시 체크리스트
- `test-plan-ko.md`: 모바일 QA 테스트 플랜
- `sample-review-pdf.txt`: 심사용 샘플 PDF 원문

현재 앱 셸은 Capacitor 기반이며 Bundle ID 후보는 `app.doclisten.mobile`입니다.
