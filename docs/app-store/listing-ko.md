# DocListen 앱스토어 등록 문구 초안

## 앱 이름
DocListen

## 부제
PDF를 눈으로만 읽지 말고 귀로 들으세요

## 카테고리 후보
- 생산성
- 교육
- 참고자료

## 짧은 설명
긴 PDF 문서를 업로드하고 문단별로 선택해 자연스러운 한국어 음성으로 들을 수 있는 PDF 듣기 앱입니다.

## 상세 설명
DocListen은 강의자료, 리포트, 논문, 업무 문서처럼 긴 PDF를 읽어야 하는 사람을 위한 PDF 음성 청취 앱입니다. PDF를 업로드하면 문서 화면을 보면서 원하는 문단을 선택하고, 현재 위치부터 자연스럽게 들을 수 있습니다.

주요 기능:
- PDF 업로드 및 페이지 보기
- 문단 터치로 원하는 위치부터 듣기
- 현재 읽는 문단 하이라이트
- 듣기, 일시정지, 이전/다음 문단 이동
- 음성 속도 조절
- 무료 체험 사용량 안내
- 파일 처리 정책과 개인정보 안내 제공

추천 대상:
- 출퇴근길에 문서를 듣고 싶은 사용자
- 긴 리포트/논문을 반복 확인해야 하는 학생·연구자
- 업무 자료를 눈으로만 보기 어려운 직장인
- PDF 문서를 접근성 관점에서 듣고 싶은 사용자

주의:
텍스트 선택이 가능한 PDF에 가장 적합합니다. 스캔 이미지 PDF나 복잡한 표 중심 문서는 음성 품질이 떨어질 수 있습니다.
