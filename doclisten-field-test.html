DocListen 비공개 테스트 기록: 실제 PDF 업로드부터 문단 듣기까지

작성·검토: DocListen 운영팀 · 최초 공개: 2026년 07월 16일 · 최종 검토: 2026년 7월 16일

이 글은 홍보 문구가 아니라 Google Play 비공개 테스트용 Android 빌드에서 실제 샘플 PDF를 열어 본 기록입니다. 확인한 범위와 확인하지 못한 범위를 나눠 적었습니다. 홈페이지의 로그인 없는 샘플 체험에서도 같은 1페이지 샘플을 직접 열어볼 수 있습니다.

테스트 범위

Google Play 비공개 테스트용 Android 빌드, 직접 만든 1페이지 텍스트형 PDF, 재생 속도 1.0배를 사용했습니다. 샘플에는 개인정보·저작권 자료가 없습니다.

1. 앱을 열었을 때 확인한 것

첫 화면에는 PDF 업로드 버튼, 사용가이드, 공개 정책과 문의 경로가 보입니다. Android 앱에서는 로그인 없이 파일 선택과 문단 듣기를 시작할 수 있도록 구성했습니다. 웹에서는 무료 사용량 관리를 위해 일반 문서 업로드 전에 Google 로그인을 요구하지만, 공개 샘플은 로그인 없이 확인할 수 있습니다.

2. 샘플 PDF를 올린 뒤

테스트 파일은 제목과 세 개의 짧은 문단으로 만든 단일 열 PDF입니다. 업로드 후 파일명이 플레이어에 표시되고, 듣기·일시정지·이전 문단·다음 문단·속도 조절이 나타나는지 확인했습니다. 이 단계에서는 아직 음성이 재생되지 않았기 때문에 일시정지 버튼이 비활성화된 것도 화면과 일치합니다.

3. 문단을 눌러 실제로 들었을 때

문단을 선택하자 PDF 화면의 해당 줄에 초록색 하이라이트가 표시됐고, 플레이어에도 같은 문장이 나타났습니다. 페이지 표시는 1/1, 속도는 1.0배였습니다. 이 확인으로 단일 열 텍스트 PDF에서 페이지 렌더링, 문단 선택, 현재 문단 표시, 재생 조작부가 하나의 흐름으로 연결되는 것을 검증했습니다.

직접 재현하는 방법

  1. 홈페이지에서 로그인 없이 샘플 체험을 누릅니다.
  2. 샘플 PDF가 열리면 첫 문단과 페이지 수를 확인합니다.
  3. 문단을 누르거나 듣기 버튼을 누릅니다.
  4. 초록색 하이라이트와 플레이어 문장이 같은지 확인합니다.
  5. 이전·다음 문단과 속도 조절을 시험합니다.

공개 샘플은 서비스 흐름을 보여주기 위한 자체 제작 자료입니다. 사용자의 실제 문서를 대신하지 않으며, 샘플 체험 과정에서 개인 PDF를 서버에 전송하지 않습니다. 실제 문서를 사용할 때는 파일 처리 정책과 허용·금지 콘텐츠 기준을 먼저 확인해 주세요.
