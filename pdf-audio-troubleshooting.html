PDF 음성이 안 나올 때 확인할 것

작성/검토: DocListen 운영팀 · 최종 업데이트: 2026년 7월 4일

PDF 음성 서비스는 브라우저, 파일 구조, 네트워크 상태의 영향을 받습니다. 소리가 나오지 않거나 문단이 이상하게 읽히면 아래 항목을 차례대로 확인해보세요.

1. 브라우저 확인

모바일에서는 Chrome 또는 Samsung Internet을 권장합니다. 오래된 브라우저나 앱 내부 브라우저에서는 파일 선택이나 음성 재생이 제한될 수 있습니다.

2. 파일 형식 확인

파일 확장자가 PDF인지, 용량이 너무 크지 않은지, 텍스트 선택이 가능한 문서인지 확인합니다. 스캔 이미지 PDF는 문단 추출이 되지 않거나 품질이 낮을 수 있습니다.

3. 소리 설정 확인

기기 볼륨, 무음 모드, 블루투스 이어폰 연결, 브라우저 자동 재생 제한을 확인합니다. 처음 재생할 때는 사용자가 직접 듣기 버튼을 눌러야 할 수 있습니다.

4. 네트워크 확인

서버 음성 생성은 네트워크 상태에 영향을 받습니다. 연결이 불안정하면 잠시 후 다시 시도하거나 브라우저 음성 대체 재생을 확인하세요.

5. 문의할 때 필요한 정보

  • 사용 기기와 브라우저
  • 문제가 생긴 PDF 유형
  • 오류가 발생한 페이지
  • 소리가 전혀 안 나는지, 특정 문단만 이상한지
