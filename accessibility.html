DocListen 접근성 안내

작성/검토: DocListen 운영팀 · 최종 업데이트: 2026년 7월 4일

DocListen은 긴 PDF를 눈으로 계속 읽기 어려운 사용자를 위해 음성 청취 경험을 제공하는 서비스입니다. 완전한 접근성 지원을 목표로 계속 개선하고 있으며, 현재 제공되는 사용성 기준과 한계를 아래에 정리합니다.

모바일 사용성

Chrome과 Samsung Internet에서 사용하기 쉽도록 큰 버튼, 명확한 재생 컨트롤, 홈 화면 추가 안내를 제공합니다. 작은 화면에서도 PDF 업로드와 듣기 버튼을 찾기 쉽도록 UI를 구성합니다.

음성 청취 지원

문단 단위 듣기, 이전/다음 문단 이동, 속도 조절, 현재 문단 하이라이트를 제공해 긴 문서를 나눠 들을 수 있도록 돕습니다. 단, PDF 구조가 복잡하면 문단 순서가 어색해질 수 있습니다.

키보드와 화면 읽기 도구

주요 링크와 버튼은 HTML 기본 요소를 사용해 브라우저 접근성을 해치지 않도록 구성합니다. PDF 캔버스 내부 텍스트와 화면 읽기 도구 연동은 문서 구조에 따라 제한될 수 있어 계속 개선이 필요합니다.

색상과 가독성

어두운 배경과 대비가 높은 텍스트 색상을 사용하고, 링크와 버튼은 테두리와 배경색으로 구분합니다. 문단 하이라이트는 현재 재생 위치를 빠르게 찾기 위한 보조 표시입니다.

개선 요청

접근성 문제를 발견하면 사용 기기, 브라우저, 문제가 발생한 화면을 문의 페이지로 보내주세요. 실제 사용 환경의 제보를 우선 반영합니다.
