전자책과 유료 PDF를 올리기 전에 확인할 권리와 개인정보

작성·검토: DocListen 운영팀 · 최초 공개: 2026년 07월 04일 · 최종 검토: 2026년 7월 16일

전자책이나 유료 PDF를 구매했다고 해서 그 파일을 외부 서비스에 업로드할 권한까지 자동으로 생기는 것은 아닙니다. DocListen에 파일을 넣기 전에는 저작권뿐 아니라 판매처 약관, 회사·학교 규정, 개인정보 포함 여부를 함께 확인해야 합니다.

열람 권한과 외부 처리 권한은 다릅니다

구매자는 보통 개인적으로 읽을 권리를 얻지만 복제·변환·외부 서버 처리·공유까지 허용받는 것은 아닙니다. DRM이 적용된 전자책을 우회하거나, 파일을 추출해 올리는 방식은 사용하지 마세요. 이용 조건이 불분명하면 판매처나 권리자에게 먼저 확인합니다.

상대적으로 확인이 쉬운 자료

  • 본인이 직접 작성한 문서
  • 저자가 업로드와 변환을 허락한 자료
  • 퍼블릭 도메인 문서
  • 외부 도구 처리를 허용하는 공개 라이선스 자료
  • 조직 보안 규정이 명시적으로 허용한 업무 문서

공개 라이선스도 출처 표시, 변경 표시, 비영리 조건처럼 서로 다른 의무가 있으므로 라이선스 원문을 확인해야 합니다.

업로드 전에 개인정보를 제거합니다

주민등록번호, 계좌번호, 고객명단, 환자정보, 학생정보, 미공개 계약 조건이 포함된 PDF는 사용하지 않는 것이 원칙입니다. 꼭 필요한 경우에는 민감한 페이지와 메타데이터를 제거한 별도 사본을 만들고, 조직의 보안 담당자에게 외부 처리 가능 여부를 확인하세요.

DocListen의 파일 처리 범위

PDF는 페이지 표시와 문단 추출을 위해 브라우저 또는 앱에서 열립니다. 사용자가 재생을 요청한 문장은 음성 생성을 위해 처리될 수 있습니다. 구체적인 저장·캐시·삭제 기준은 파일 처리 정책에, 계정과 개인정보 기준은 개인정보처리방침에 공개합니다.

판단이 어려울 때 사용할 체크리스트

  1. 내가 이 문서를 만들었거나 이용 허락을 받았는가
  2. 판매처 약관이 외부 서비스 처리를 금지하지 않는가
  3. DRM이나 접근 제한을 우회하지 않았는가
  4. 개인정보와 영업비밀을 제거했는가
  5. 결과를 다른 사람에게 공유하지 않는가

하나라도 확실하지 않다면 업로드하지 않는 편이 안전합니다. 이 페이지는 일반적인 서비스 이용 기준이며 개별 저작권 분쟁에 대한 법률 자문은 아닙니다.

권리 침해 신고와 삭제 요청

권리자가 DocListen과 관련된 문제를 발견하면 문의 페이지에서 대상 URL과 권리 관계를 알려줄 수 있습니다. 계정과 데이터 삭제 절차는 계정 삭제 안내에 공개합니다. 신고 과정에서도 원본 신분증이나 불필요한 민감정보를 보내지 마세요.
