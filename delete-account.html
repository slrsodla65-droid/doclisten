DocListen 계정 및 데이터 삭제

운영: DocListen 운영팀 · 최종 업데이트: 2026년 7월 13일

DocListen의 Google 로그인 계정과 관련 사용량 기록은 앱 안에서 직접 삭제하거나 고객지원 채널을 통해 삭제 요청할 수 있습니다. 앱을 기기에서 삭제하는 것만으로는 서버 계정 기록이 삭제되지 않습니다.

방법 1. 앱에서 직접 삭제

  1. DocListen에서 Google 계정으로 로그인합니다.
  2. 회원 영역의 계정 삭제 버튼을 누릅니다.
  3. 삭제 확인 창에서 삭제를 승인합니다.
  4. 완료 안내가 표시되면 서버 계정 기록이 삭제되고 현재 기기에서도 로그아웃됩니다.

방법 2. 웹에서 삭제 요청

  1. DocListen 고객지원 채널을 엽니다.
  2. 메시지에 “DocListen 계정 삭제 요청”이라고 적습니다.
  3. 계정 확인을 위해 Google 로그인 이메일만 전달합니다. 비밀번호, 인증코드, 결제정보 또는 PDF 원문은 보내지 마세요.
  4. 운영팀이 계정을 확인한 뒤 영업일 기준 7일 이내에 처리 결과를 안내합니다.

삭제되는 데이터

  • Google 로그인 이메일
  • DocListen 서비스 인증 토큰
  • 사용자 플랜과 일별 문단 사용량 기록
  • 계정에 직접 연결된 서비스 권한 기록

별도로 삭제 요청할 수 있는 데이터

계정을 유지한 상태에서도 고객지원 채널을 통해 파일 처리 기록 또는 식별 가능한 음성 캐시의 삭제를 요청할 수 있습니다. 요청 시 대략적인 이용 날짜와 짧은 문구만 알려주세요. PDF 원문 전체나 민감정보는 보내지 마세요.

보관될 수 있는 데이터와 기간

  • 개인을 식별하지 않는 집계 서비스 지표는 계정 삭제 후에도 통계 목적으로 보관될 수 있습니다.
  • 보안 및 장애 대응을 위한 기본 접속·오류 기록은 최대 90일 보관된 후 삭제될 수 있습니다.
  • 법령상 보관 의무가 있는 정보가 있다면 해당 법정 기간 동안만 보관합니다.

문의

삭제 요청 처리나 개인정보에 관한 문의는 DocListen 문의 페이지에서 접수할 수 있습니다.
